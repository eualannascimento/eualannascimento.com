# Pendências de certificados

Arquivo de controle interno (não publicado no site) para certificações/cursos adicionados sem nenhuma evidência (nem imagem, nem link de credencial). Ver Regra 08 de `.docs/specs/redesign-site-pessoal.md`.

Nenhuma pendência no momento: as 8 certificações novas trazidas do LinkedIn em 2026-07-20 (Power Platform Fundamentals, Generative AI Fundamentals, Power BI Data Analyst, Azure Data Fundamentals, Azure Fundamentals, Notion Workflows, Notion Essentials, Databricks Fundamentals) têm link de credencial verificável e foram listadas em `certif.html` com o marcador 🔗.

## Como usar
Ao adicionar uma certificação/curso sem nenhuma evidência disponível, registre aqui:

- [ ] Nome do curso/certificação - instituição - data - o que falta (print do certificado, link de verificação, etc.)
